
Remove sensitive data from the EC2 instance user-data generated by launch templates

```hcl
resource "aws_launch_template" "good_example" {
  image_id      = "ami-12345667"
  instance_type = "t2.small"
  user_data     = <<EOF
	 export GREETING=hello
EOF
}
```
